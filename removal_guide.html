How to Remove the Annoying Pop-Up Script

Warning: This guide is for advanced users only. Incorrect modifications can cause unexpected issues. Always backup your files before making changes.

Step 1: Open the System Monitoring Center

First, open the System Monitoring Center. You can usually find this in your system's menu under System Tools or by searching for "System Monitor".

Step 2: Navigate to the Processes Tab

Once the System Monitoring Center is open, go to the Processes tab. This tab shows a list of all running processes on your system.

Step 3: Identify the Pop-Up Script

Look through the list of processes to find the one named pop_up.sh. This is the script causing the pop-ups.

Step 4: Kill the Pop-Up Script

Once you've identified the process, right-click on it and select Kill or End Process. This will stop the script and remove the pop-ups.

Note: If you are unable to find the pop_up.sh process, try sorting the processes by CPU usage or memory usage to locate the script more easily.

Alternative Solution: Use Command Line

If you prefer using the command line, open a terminal and type the following command to find the process ID (PID) of the script:

ps aux | grep pop_up.sh

Then, use the kill command followed by the PID:

kill [PID]
